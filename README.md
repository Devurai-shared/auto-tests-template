## Playwright Sample Project

#### Test Services

https://demo.learnwebdriverio.com

#### Requirements

- NodeJS
- GIT

#### Install project dependencies

```bash
npm install
```

#### Run E2E Tests

```bash
npm run test:e2e
```

#### Run Visual Tests

```bash
npm run test:visual
```

#### Run Accessibility Tests

```bash
npm run test:a11y
```

#### Run API Tests

```bash
npm run test:api
```

#### Generate HTML Report

```bash
npm run html:report
```
